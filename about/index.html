ManaLAB

L'atelier Magic francophone — collection · decks · analyse · communauté

Pourquoi ManaLAB ?

Les outils de gestion de Magic: The Gathering existent depuis longtemps, mais aucun n'a été pensé pour les joueurs francophones :

  • Moxfield et Archidekt sont US-only, en anglais, sans intégration Cardmarket EU
  • EDHREC est une base de données précieuse mais à sens unique, sans social ni gestion de collection
  • ManaBox est mobile-only, sans analyse de deck poussée ni communauté

ManaLAB a été conçu pour combler ce vide : un outil complet, en français d'abord, centré sur la communauté de joueurs.

Ce que ManaLAB fait de mieux

📊
Analyse de deck profonde
Bracket Commander 1-5 auto-détecté, heatmap par tour de jeu, détection de combos infinis, anti-synergies, coaching textuel.
🤝
Cross-collection
Vois quelles cartes tes amis possèdent déjà — quand tu construis un deck, les cartes disponibles dans le groupe sont mises en valeur.
📷
Scan & édition exacte
Photographie tes cartes au téléphone, ManaLAB reconnaît la version exacte (set + numéro de collection).
⚔
Arène en ligne
Jouez à plusieurs sur un plateau virtuel avec gestion des règles, historique, mulligan London, draft et scellé multijoueur.
🌐
Decks publics & follows
Partage tes meilleures listes, suis d'autres deckbuilders, like et commente. Une vraie communauté MTG en français.
🛒
Listes Cardmarket partagées
Organise des commandes groupées avec tes amis, suis les arrivages, marque ce qui est reçu.

Le projet en bref

ManaLAB est développé en solo par Thibaud Combes, joueur de Magic depuis des années et développeur. Le projet est gratuit, sans publicité, sans tracking publicitaire. Les seuls revenus envisagés sont :

  • Un abonnement ManaLAB Pro optionnel (~3€/mois) pour les utilisateurs qui veulent soutenir le projet et débloquer quelques fonctionnalités avancées (analyse IA, alertes prix illimitées, customisation profil)
  • Une affiliation discrète avec Cardmarket sur les liens d'achat — ce qui ne change rien au prix pour toi

L'app gratuite restera toujours largement utilisable. C'est mon outil personnel — je veux qu'il soit utile à tout le monde.

"Magic c'est un jeu social. L'outil qui le gère devrait l'être aussi."

Stack technique

  • Application web monolithique (single-file HTML/CSS/JS vanilla) — pas de framework lourd, pas de build complexe
  • PWA installable sur mobile et desktop
  • Backend Firebase (Firestore en région EU pour RGPD)
  • Edge functions Vercel pour SEO et notifications
  • Données de cartes : Scryfall API (avec leur accord)
  • Code dépôt privé pour l'instant — un open-source partiel est envisagé

Confidentialité & RGPD

ManaLAB ne vend ni ne partage tes données. Les données sont stockées chez Firebase en région européenne. Aucun cookie publicitaire, aucun tracker tiers, aucun cookie d'identification (Plausible Analytics est utilisé en mode anonyme).
